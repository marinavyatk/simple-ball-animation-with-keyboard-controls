# Simple ball animation with keyboard controls
Keyboard control:  
:arrow_right: - right  
:arrow_left: - left  
:arrow_up: - up  
:arrow_down: - down  
V - increase the size of the ball  
C - reduce the size of the ball  
X - increase speed  
Z - decrease speed  
You can also set the speed using the keys from 1 to 9.
